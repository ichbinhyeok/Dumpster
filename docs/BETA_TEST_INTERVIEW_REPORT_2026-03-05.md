## Confirmed Risks

- Weak/overlapping pages still require strict noindex/merge discipline.
- Price/booking expectation can diverge from "decision support" positioning if CTA copy is vague.
- Provider variance must remain explicit for weight-limit and heavy-material guidance.

## Execution Guidance

1. Keep index scope constrained to the current priority set.
2. Prefer evidence density over URL expansion.
3. Keep copy in user query language, not internal product language.
4. Maintain quick-first UX and expose advanced controls only when needed.
5. Treat confidence/variance disclosure as mandatory, not optional.
